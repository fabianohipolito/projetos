## Devops

O objetivo deste repositorio é criar e gerenciar servidores utilizando feramentas Devops e de monitoramento.

## Ferramentas Devops

- Vagrant
- Docker
- Ansible
- Rundeck
- Jenkins
- Git

## Ferramentas de monitoramento DevOps

- Prometheus
- Grafana
- Zabbix
- Graylog
